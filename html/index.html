Course Roadmap: Day by Day

Day 1: HTML Basics

Structure: <!DOCTYPE>, <html>, <head>, <title>, <body>. Comments: <!-- comment -->.

Day 2: Headings & Paragraphs

<h1>-<h6>, <p>, <br>, <hr>, text formatting tags: <b>, <strong>, <i>, <em>, <mark>, <sub>, <sup>, <small>.

Day 3: Links & Images

<a> links, <img> with alt, <code> <pre> for snippets.

Day 4: Lists

<ul>, <ol>, <li> and nested lists.

Day 5: Mini Project

“About Me” page combining all tags learned so far.

Days 6–8: Page Structure

<div>, semantic tags: <header>, <footer>, <section>, <article>, inline vs block elements, <span>.

Days 9–11: Tables & Quotes

<table>, <tr>, <th>, <td>, <caption>, <blockquote>, <q>, <cite>.

Days 12–15: Forms & Input

<form>, <input> types: text, email, password, number, radio, checkbox, <textarea>, <select>. Form attributes: placeholder, required, maxlength, action, method.

Days 16–18: Media & Embeds

<img> advanced, <audio>, <video>, <iframe> for YouTube & Google Maps.

Days 19–22: Styling Basics

Inline CSS, internal CSS with <style>, colors, fonts, backgrounds, borders, hover effects.

Days 23–26: Advanced Styling

CSS animations, transitions, flexbox, grid layouts, responsive design.

Days 27–30: Capstone Project

Build a full portfolio page with header, bio, skills, achievements table, media, contact form. Use all learned concepts.
